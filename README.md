# Бюджетный трекер

Приложение, которое позволяет отслеживать свои финансовые траты и доходы. 
Пользователи могут добавлять свои транзакции, просматривать статистику и диаграммы расходов.

Приложение состоит из трех табов: главная, транзакции и аналитика.

В разделе транзакций пользователь может добавлять операции, по нажатию на плавующую кнопку открывается экран для заполнения данных по операции
<img width="386" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/03e4e452-d1c2-4bd8-875f-32f13d64ff6e">
<img width="371" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/78bf5410-5a8c-4b15-8120-d6a6382cb502">

После добавления транзакций они отображаются списком с группировкой по месяцам и годам. Также на экране списка операций есть поиск и фильтры
<img width="378" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/5b302b4d-c78c-4863-97bd-1a43fc7eb477">

При тапе на транзакцию открывется экран для редактирования с возможностью полного удаления
<img width="377" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/5a75e7af-f7da-408f-bee7-70cc65002052">

По нажатию на текстовое окно поиска открывается отдельный экран, в котором будут выводиться результаты поиска
<img width="371" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/99d8233c-7fdf-47df-a534-05b938036aa7">

По тапу на иконку фильтров открывается экран для выбора фильтров. Цвет иконки зависит от того, есть ли выбранные фильтры.
<img width="391" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/90835439-1ac9-48f2-bcbc-09720478138d">

На странице аналитики отображаются графики, первый - расходы и доходы по месяцам за последний год, второй - траты по категориям
<img width="401" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/ebe3bf41-ba9a-4174-a24b-98a12728e4e9">

На главной странице отображаются суммы расходов и доходов, последние операции, лимиты
<img width="384" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/5c4148f0-9cff-47ee-9ff9-b11f3cb30a4d">

При нажатии на компонент с информацией о лимитах открывается экран со списком всех доступых категорий, по тапу на одну из них можно установить лимит
<img width="384" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/04db5df0-68cc-453b-acf8-045b59937dce">
<img width="415" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/3e8d953c-4b69-4221-9868-047fabf588d2">

После установки лимитов, категории с установленными лимитами будут отображаться на главной
<img width="355" alt="image" src="https://github.com/RedFox802/budget_tracker_app/assets/92377503/0f6ca12c-30bb-4b38-b00f-6f50387d83d6">

При превышении лимитов или приближению к ним отображается предупреждение на главном экране
